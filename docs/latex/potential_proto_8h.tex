\doxysubsection{Detailed Description}
Prototype header for automatically generated scalar field potential and derivative calculations. 

This file serves as a template that {\ttfamily Py\+Trans\+Setup.\+py} uses to generate {\ttfamily \doxylink{potential_8h_source}{potential.\+h}}. It defines the function signatures for evaluating the potential, its gradient, Hessian, and third derivatives. The actual implementations are symbolically derived and inserted by the Python setup script.
