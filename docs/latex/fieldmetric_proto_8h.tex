\doxysubsection{Detailed Description}
Prototype header for automatically generated field space metric and curvature calculations. 

This file serves as a template that {\ttfamily Py\+Trans\+Setup.\+py} uses to generate {\ttfamily \doxylink{fieldmetric_8h_source}{fieldmetric.\+h}}. It defines the function signatures for computing the field space metric, Christoffel symbols, Riemann tensor, and its covariant derivative. The actual implementations are symbolically derived and inserted by the Python setup script.
