\doxysubsection{Detailed Description}
Represents the 2-\/point correlation function for tensor perturbations. 

Implementation of the Tensor two-\/point correlation function.

This class handles the initialization and provides methods related to the 2-\/point correlation function for tensor modes, denoted as \doxylink{class_gamma}{Gamma}.
