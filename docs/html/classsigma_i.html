Detailed Description

Represents the imaginary part of the 2-point correlation function for scalar perturbations (Sigma).

Implementation of the IMAGINARY phase-space two-point correlation function.

This class is similar to sigma but specifically handles the imaginary component of the 2-point function, which can arise from certain quantum mechanical treatments or specific initial states.
