Detailed Description

Represents the 3-point Multi-Point Propagator (MPP3) tensor.

Instance of the second MPP matrix.

This class manages the initialization of the MPP3 tensor, which is used in the Multi-Point Propagator formalism for computing 3-point correlation functions. It also computes the evolution tensor M for Rho2.
