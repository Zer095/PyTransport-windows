Detailed Description

Represents the 2-point correlation function for scalar perturbations (Sigma).

Implementation of the REAL phase-space two-point correlation function.

This class handles the initialization of the 2-point correlation function (Sigma) for a given wavenumber k and sets up the initial conditions for its evolution based on the background fields and parameters. It also computes the evolution matrix M for Sigma.
