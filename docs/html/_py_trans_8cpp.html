Detailed Description

Core C++ implementations for the PyTransport 3.0 module.

This file defines the C++ functions that are exposed to Python through the PyTransport module. It provides the numerical backbone for computing primordial correlators in inflationary cosmology, implementing both the standard Transport formalism and the Multi-point propagator (MPP) approach.

Introduction

PyTransport constitutes a straightforward code written in C++ together with Python scripts which automatically edit, compile and run the C++ code as a Python module. It has been written for Unix-like systems (OS X and Linux). PyTransport relies on the Transport formalism, which implements a differential formalism to compute inflationary correlation functions. It is able to compute tree-level correlators for multi-field models with canonical and non-canonical field space. Along with the standard Transport formalism, PyTransport 3.0 also implements the Multi-point propagator (MPP) approach to inflationary correlators. The MPP approach provides an alternative way to compute two- and three-point correlation functions, and related observables, with advantages and limits discussed in related literature.
