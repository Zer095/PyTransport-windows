Detailed Description

Represents the 3-point correlation function (Alpha).

Implementation of the REAL phase-space three-point correlation function.

This class handles the initialization of the 3-point correlation function (Alpha) for a triplet of wavenumbers (k1, k2, k3) and sets up the initial conditions for its evolution based on the background fields and parameters.
